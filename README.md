# Smart Task Manager

A responsive task management web application built using React.js, Node.js and Express.js.

## Features

- Add Tasks
- Edit Tasks
- Delete Tasks
- Mark Tasks as Completed
- Local Storage Support
- Responsive UI

## Technologies Used

- React.js
- CSS3
- JavaScript
- Node.js
- Express.js

## Run Project

Frontend:

cd client
npm install
npm run dev

Backend:

cd server
npm install
node server.js
